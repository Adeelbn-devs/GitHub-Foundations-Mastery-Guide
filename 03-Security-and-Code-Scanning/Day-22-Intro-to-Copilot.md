# Day 22: Introduction to GitHub Copilot 🤖

**Date:** May 18, 2026  
**Module:** GitHub Copilot

## 📝 What I Learned Today:

Today, I started the highly anticipated module on **GitHub Copilot**, the AI-powered developer assistant. 

### Key Concepts:
- **The AI Pair Programmer:** Copilot uses the context of your existing code and comments to generate code, explain complex logic, refactor functions, and write tests.
- **The Engine:** It is powered by **OpenAI Codex**, a generative pretrained language model designed specifically for software development.
- **The Environment:** Copilot doesn't run in the browser; it integrates directly into your workflow via extensions for IDEs like **VS Code**, Visual Studio, Neovim, and JetBrains.

Moving forward, I will be learning how to properly trigger Copilot, understand the differences between its billing tiers (Free, Pro, Business, Enterprise), and how to configure and troubleshoot the extension.
