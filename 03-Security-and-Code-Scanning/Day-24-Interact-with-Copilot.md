# Day 24: Interacting with GitHub Copilot ⌨️🤖

**Date:** May 26, 2026  
**Module:** GitHub Copilot

## 📝 What I Learned Today:

Today, I explored the practical ways to trigger and interact with GitHub Copilot directly within the IDE (Visual Studio Code). Knowing these shortcuts is essential for maximizing productivity.

### Core Interaction Methods:
1. **Inline Suggestions:** The most common way Copilot works. It predicts the next lines of code in *greyed-out* text.
   - **Accept:** Press `Tab`
   - **Reject:** Press `Esc`
2. **Comments to Code:** Writing a natural language comment (e.g., `# Function to reverse string`) prompts Copilot to generate the actual implementation.
3. **Multiple Alternatives:** If the first suggestion isn't ideal, pressing `Alt + ]` cycles through different coding approaches.

### Copilot Chat & Slash Commands:
Using `Ctrl + I` opens the inline chat where you can use powerful **Slash Commands**:
- `/explain`: Generates a natural language explanation of a selected, complex block of code.
- `/tests`: Automatically generates unit test cases for a selected function, ensuring code reliability.
- `/suggest`: Offers improvements or refactoring suggestions based on the current context.
