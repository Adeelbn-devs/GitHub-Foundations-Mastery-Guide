# Day 10: GitHub Mobile and GitHub Desktop 🚀

**Date:** May 10, 2026  
**Module:** Introduction to GitHub Administration

## 📝 What I Learned Today:

Today, I explored alternative, highly efficient ways to interact with GitHub beyond the standard web browser and command-line interface.

### 1. GitHub Mobile
A first-party client for iOS and Android that allows developers to manage work on the go.
- **Use Cases:** Triaging notifications, reviewing and merging pull requests, and performing lightweight file edits away from the keyboard.
- **Security:** Acts as a secure tool for Two-Factor Authentication (2FA) and verifying sign-in attempts.

### 2. GitHub Desktop
An open-source GUI application for Windows and macOS designed to streamline Git workflows.
- **Use Cases:** Ideal for users who prefer visual interfaces over the CLI.
- **Key Features:**
  - Visually review code diffs and even compare changed image files.
  - Easily clone repositories and manage local projects.
  - Interactively select changes to commit and quickly add co-authors.
  - View Continuous Integration (CI) statuses directly in the app.
